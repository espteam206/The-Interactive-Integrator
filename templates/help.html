Concrete Calculator Help

Overview

Use this page to calculate concrete material requirements and view a sunburst chart for composition.

Instructions

  • Enter material amounts as numeric values: cement, sand, aggregate, water, admixture.
  • Use the Add/Remove row controls to adjust SCM/Aggregate/Admixture entries.
  • Fill route addresses only if you want distance estimates per component row.
  • Click Calculate to run the calculation and render the diagram.

Note

All fields in each active row must be completed to submit successfully.
