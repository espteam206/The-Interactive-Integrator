# The-Interactive-Integrator
A website written using Python and Flask that calculates concrete carbon footprint for A1-A2 via API calls to EC3 and Google Maps and outputs a Sunburst Diagram. Vibecoded.

API Calls:
https://mapsplatform.google.com/lp/maps-apis/

https://www.buildingtransparency.org/tools/

Python Framework:
https://flask.palletsprojects.com/en/stable/

https://plotly.com/python/sunburst-charts/
